# Restaurant
Application backend for contactless customer service in restaurant.

## Technology stack
* Spring Boot
* Spring Web
* Spring Data
* Spring Security
* Springdoc OpenAPI
* ModelMapper
* JJWT
* Lombok
* MySQL

## Setup
1. Clone repository

<pre>git clone https://github.com/jchojdak/restaurant.git</pre>

2. Open the cloned directory

<pre>cd restaurant</pre>

## OpenAPI 3.0
### Swagger
